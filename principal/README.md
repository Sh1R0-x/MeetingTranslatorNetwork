# MeetingTranslatorNetwork — Web Dashboard (Prototype)

Experimental React/TypeScript dashboard for MeetingTranslatorNetwork.

> **Note**: This is a prototype UI and is not currently integrated with the main PyQt6 desktop application.

## Setup

**Prerequisites**: Node.js

1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the dev server:
   ```bash
   npm run dev
   ```
